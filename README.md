# huawei-wmi

Sets group write privileges & reinstates battery charging-thresholds.

## Debian Package

Tested on Debian Buster.

Requirements:
  * `build-essential`
  * `devscripts`
  * `debhelper`

Create: `make`
Install: `sudo make install`
Uninstall: `sudo make uninstall`
Reconfigure: `sudo make reconfigure`
