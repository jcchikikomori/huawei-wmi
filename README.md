# huawei-wmi

Sets group write privileges and reinstates battery charging-thresholds.

## Debian Package

Tested on Debian Buster.

Requirements:
  * `build-essential`
  * `devscripts`
  * `debhelper`

Create: `make`
Install: `sudo make install`
remove: `sudo make remove`
Reconfigure: `sudo make reconfigure`
Clean: `make clean`
Distclean: `make distclean`
